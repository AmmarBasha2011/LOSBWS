# 🚀 Ultimate Local OS Bridge (LOSBWS V5)

The **Ultimate Local OS Bridge** is a high-performance, secure FastAPI-based tool designed to give AI agents (like OpenClaw, Anthropic, or custom bots) full control over a Linux environment. It provides over 50+ endpoints for filesystem management, system monitoring, network diagnostics, and data processing.

---

## 🌟 Key Features

### 📁 Advanced Filesystem Control
- **Standard Ops:** Create, Read, Update, Delete (CRUD) for files and directories.
- **Bulk Operations:** Split large files, merge chunks, and recursive directory emptying.
- **Search:** Grep-like search across directories and Regex-based file content matching.

### 💻 System & OS Power
- **Process Management:** List processes (sorted by CPU/MEM) and kill them by PID.
- **Terminal Access:** Execute any shell command directly through the bridge.
- **Git Integration:** Check repository status and manage local codebases.

### 🌐 Network & Security
- **Proxy Capabilities:** Perform `curl` requests from the host machine to bypass local restrictions.
- **Diagnostics:** Check open ports, public IP, and download files directly from the web.
- **Integrity:** Generate SHA256/MD5 hashes and handle Base64 encoding/decoding.

### 📦 Compression & Archives
- Full support for `tar`, `untar`, `gzip`, and `gunzip` for efficient data movement.

## 🛠️ Installation & Setup

### Prerequisites
- Python 3.10+
- Linux Operating System (Ubuntu/Debian recommended)
- `pip` and `venv`

### Step-by-Step Installation
1. **Clone the Repository:**
   ```bash
   git clone https://github.com/AmmarBasha2011/LOSBWS
   cd LOSBWS
   ```

2. **Run the Setup/Execution Script:**
   The project includes a `run.sh` script that handles virtual environment activation and dependency checks.
   ```bash
   chmod +x run.sh
   ./run.sh
   ```

3. **Obtain the Public Link:**
   Once launched, a **Tkinter GUI popup** will appear with your unique Cloudflare Tunnel URL. Click **"Copy Link & Close"** to save it to your clipboard.

- **Safety First:** Restricts AI to the `workdir/` sandbox.
- **Dynamic:** Handles the changing Cloudflare URLs automatically.

---

## 🔒 Security Architecture

- **Sandbox Environment:** All file operations are restricted to the `workdir/` directory. Attempts to use `../` to escape the sandbox will result in a `403 Forbidden` error.
- **API Key Protection:** Every request requires an `api_key` parameter. Default is `ammar123`.
- **Real-time Logging:** Every request made by the AI is logged in the terminal with status codes and processing time.
